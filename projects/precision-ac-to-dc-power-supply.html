Project

Precision AC to DC Power Supply

Analog Electronics, Voltage Regulation, Current Limiting, and PCB Design

Completed / Functional Coursework Project

Short Description

A regulated AC/DC input power supply designed to produce a stable 7 V DC output with current limiting around 150 mA. The project began as a class final, but was refined beyond the base requirements through component selection, tuning, testing, and PCB cleanup.

Project Overview

This project was originally assigned as an AC to DC power supply final. Rather than only meeting the minimum requirements, I wanted to see how closely I could target a precise output voltage by carefully selecting components, validating values, and keeping the PCB implementation clean.

The design accepts either AC or DC input in the 13–30 V range and regulates it to approximately 7 V DC. The circuit includes rectification, filtering, a voltage reference, op-amp regulation, MOSFET output control, LED indication, and current limiting.

After the class report was completed, I continued refining and tuning the circuit. By replacing and selecting components more carefully, I was able to get the realized output voltage very close to the calculated target.

Features

  • Accepts AC or DC input from approximately 13–30 V.
  • Regulated 7 V DC output target.
  • Current limit near 150 mA for overload protection.
  • Input rectification and filtering stage.
  • LM385 2.5 V voltage reference.
  • Op-amp controlled MOSFET output regulation.
  • BJT-assisted current limiting feedback.
  • Red LED power/operational indicator.
  • Measured output remained around 7.1 V through most of the intended load range.
  • Measured current limit of approximately 152 mA.

Technical Challenges

  • Precision targeting: selecting and tuning components to bring the real output voltage close to the calculated design value.
  • Regulation stability: maintaining output voltage as load current increased toward the current limit.
  • Current limiting: using the voltage drop across a sense resistor to trigger current limiting through the BJT/MOSFET control path.
  • Ripple control: filtering rectified input and reducing output ripple under load.
  • Analog feedback behavior: understanding how the op-amp, MOSFET, and BJT interacted to regulate voltage and limit current.
  • PCB implementation: keeping the board clean and practical so the physical build matched the intended circuit behavior.

Miscellaneous Notes

The measured current limit was approximately 152 mA, which was very close to the intended 150 mA limit. The circuit maintained about 7.1 V under normal load conditions before entering current limit behavior.

The project was a useful exercise in moving from simulation and calculation into physical circuit performance. Small component tolerances, real op-amp behavior, reference stability, and layout quality all affected the final result.

This was one of the projects that helped deepen my understanding of analog regulation, current limiting, feedback control, and the difference between a schematic that works in theory and a circuit that behaves well on a bench.
